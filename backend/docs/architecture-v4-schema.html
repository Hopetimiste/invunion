üîÑ Triggers Auto-Calcules

update_invoice_recovery()

Declencheur : INSERT / UPDATE / DELETE sur matches

Recalcule sur invoices :

  • recovery_percent = SUM(matched_amount) / amount_incl_vat * 100
  • status ‚Üí unpaid / partial / paid

update_counterparty_payment_analytics() v4

Declencheur : INSERT / UPDATE / DELETE sur matches

Recalcule sur counterparties (si type = 'client' ou 'both') :

  • payment_score = total_paid / total_invoiced * 100
  • avg_payment_days = AVG(transaction_date - invoice_date)
  • total_invoiced, total_paid, invoice_count
  • last_invoice_date, last_payment_date
